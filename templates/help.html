Simple Terms Viewer

This is a prototype micro-application built using Python, Flask, and MariaDB. The following features are available:

  • View Terms and Conditions reference data (select Reference Data from the menu)
  • Search for a single customer and view their current Terms (select Customer Query from the menu)

The following features are on the application backlog:

  • Connect to a production database for live data
  • Allow reference data to be refreshed from a production source
    • Include logic to prevent refreshing the data too frequently
  • Add security and form validation as appropriate based on whether the other backlog items are implemented
  • Add hyperlinks to the 'Terms and Conditions Version' cells on the refdata page to open the full Terms for review
